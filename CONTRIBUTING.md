# Contributing

Contributions are welcome only when they can be shared under the same project license.

## Contributor Certification

By submitting a contribution, you confirm that:

- you created the contribution yourself, or you have the right to submit it
- the contribution does not contain confidential or proprietary material from another party
- the contribution does not contain real tokens, passwords, internal IPs, VPN hostnames, customer data, patient data, or other sensitive data
- the contribution may be included in Kabinette Notes under the project license
- you understand that commercial resale of this software remains restricted by the license

## Security Issues

Do not open public issues for suspected vulnerabilities. Follow `SECURITY.md`.

## Privacy

Do not include real logs, screenshots, names, notes, chat messages, customer data, patient data, or production configuration in issues, pull requests, examples, or documentation.

## Scope

The project is intended for internal LAN/VPN deployments. Changes that make public-internet deployment easier should include clear security documentation and should not weaken the internal-first assumptions.
